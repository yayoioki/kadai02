[画面切り替え]
----------------------------------------------------
ウィンドウサイズの切り替えは横幅600pxに設定しています。
変更したい場合は「style.css」の下の方にある
「スマホ、タブレット用」
@media screen and (max-width: 600px)
の数値と
jsフォルダの中身「jquery.toggle.js」の
600
の数値を変更してください。

[画像の縮小表示]
----------------------------------------------------
スマホ・タブレット(もしくは小画面)での閲覧の際、
大きい画像は縮小して表示されるようになっています。

縮小表示したくない場合は、style.cssの
/* 画像の縮小表示 */ の
指定部分を全て消してください。

一部の画像のみを縮小表示したい場合は、style.cssの
/* 画像の縮小表示 */ 部分の
img{
を、
img.syukusyou{
に変更し、縮小したい画像に
<img src="○○.png" class="syukusyou">
というように、classをつけてください 。

※syukusyou の部分はなんでも構いません
※gazou でも、なんでもわかりやすいもので大丈夫です。
